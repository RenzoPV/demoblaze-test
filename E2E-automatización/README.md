# Automatización E2E – DemoBlaze

Este proyecto contiene una prueba funcional automatizada de extremo a extremo (E2E) sobre el flujo de compra del sitio web https://demoblaze.com.

El escenario validado contempla las siguientes acciones:

- Agregar dos productos al carrito de compras  
- Visualizar el contenido del carrito  
- Completar el formulario de compra  
- Finalizar el proceso de compra y validar el resultado  

## Instrucciones de ejecución

1. Abrir una consola (CMD o PowerShell).
2. Dirigirse a la ruta raíz del proyecto.
3. Ejecutar el siguiente comando:

   mvn test
